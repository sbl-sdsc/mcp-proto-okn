### Key Observations

**Skeletal features** dominate the highest associations — increased arm span (log OR 7.35), arachnodactyly (6.54), disproportionate tall stature (6.17), slender fingers (5.86), and tall stature (5.11) are all hallmark connective tissue features of Marfan syndrome.

**Ocular features** appear strongly: lens position anomaly (6.73) and ectopia lentis (5.73) both rank in the top 10, consistent with the classical diagnostic triad.

**Cardiovascular manifestations** are prominent throughout: dilated cardiomyopathy (5.03), aortic valve prolapse (4.87), aortic dissection (4.13), aortic root aneurysm (4.03), and thoracic aortic aneurysm (3.48) all appear — reflecting the life-threatening aortopathy of the disease.

**Comorbid connective tissue conditions** like Ehlers-Danlos syndrome (4.53) and osteogenesis imperfecta (4.11) also appear, suggesting frequent clinical overlap or shared genetic architecture.

### SPARQL Query Used

```sparql
PREFIX biolink: <https://w3id.org/biolink/vocab/>
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>
PREFIX xsd: <http://www.w3.org/2001/XMLSchema#>

SELECT DISTINCT ?phenotypeLabel ?phenotypeURI ?logOR ?logOR_CI ?totalSampleSize
WHERE {
  BIND(<http://purl.obolibrary.org/obo/MONDO_0007947> AS ?disease)

  ?assoc biolink:subject ?disease ;
         biolink:object ?phenotype ;
         biolink:has_supporting_studies ?study .

  ?phenotype rdfs:label ?phenotypeLabel .
  BIND(?phenotype AS ?phenotypeURI)

  ?study biolink:has_study_results ?result .
  ?result biolink:log_odds_ratio ?logOR .

  OPTIONAL { ?result biolink:log_odds_ratio_95_ci ?logOR_CI }
  OPTIONAL { ?result biolink:total_sample_size ?totalSampleSize }
}
ORDER BY DESC(xsd:decimal(?logOR))
LIMIT 100
```
